Forum Experiment

I call this an experiment because it was only ever supposed to be a way for me to test out Go. I tried to create that classic forum experience that has mostly been solved by popular software or destroyed entirely. This was also where I learned a lot about HTML Templates and definitely changed how I believe content on the web should be rendered.

Features

  • Account Creation
  • CLI Based Moderation Suite
  • Actually good reply system
